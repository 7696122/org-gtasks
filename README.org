* Org-Gtasks
Export all Google Tasks to org file.

This module has been HIGHLY inspired by [[https://github.com/myuhe/org-gcal.el][org-gcal]]


** Config

#+begin_src elisp
(require 'org-gtasks)
(org-gtasks-register-account :name "Perso"
			     :directory "~/.emacs.d/gtasks/"
                             :client-id "XXXX"
                             :client-secret "XXXX")
#+end_src
